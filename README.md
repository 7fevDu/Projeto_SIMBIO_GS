# Projeto_SIMBI_GS
Projeto acadêmico desenvolvido na FIAP em parceria com uma empresa de RH. Realizei a geração de dados em Python utilizando IA (Gemini), tratamento e análise com machine learning e validação cruzada. Em seguida, containerizei a solução com Docker, publiquei na nuvem e integrei ao Power BI para atualização automática dos dados.
